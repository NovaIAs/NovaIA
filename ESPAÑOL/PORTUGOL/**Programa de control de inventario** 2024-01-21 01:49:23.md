```portuñol
**Programa de control de inventario**

**Variable**

* inventario: Una matriz de registros que contiene información sobre cada artículo en el inventario.
* cliente: Una matriz de registros que contiene información sobre cada cliente.
* pedido: Una matriz de registros que contiene información sobre cada pedido.
* fecha_actual: La fecha actual, como un número entero en el formato AAAA-MM-DD.

**Funciones:**

* **leer_inventario()** : Lee el archivo de inventario y carga la matriz de inventario.
* **leer_clientes()** : Lee el archivo de clientes y carga la matriz de clientes.
* **leer_pedidos()** : Lee el archivo de pedidos y carga la matriz de pedidos.
* **mostrar_inventario()** : Muestra la matriz de inventario en la consola.
* **mostrar_clientes()** : Muestra la matriz de clientes en la consola.
* **mostrar_pedidos()** : Muestra la matriz de pedidos en la consola.
* **añadir_articulo()** : Añade un nuevo artículo al inventario.
* **borrar_articulo()** : Elimina un artículo del inventario.
* **editar_articulo()** : Edita un artículo en el inventario.
* **añadir_cliente()** : Añade un nuevo cliente a la lista de clientes.
* **borrar_cliente()** : Elimina un cliente de la lista de clientes.
* **editar_cliente()** : Edita un cliente en la lista de clientes.
* **añadir_pedido()** : Añade un nuevo pedido a la lista de pedidos.
* **borrar_pedido()** : Elimina un pedido de la lista de pedidos.
* **editar_pedido()** : Edita un pedido en la lista de pedidos.
* **procesar_pedido()** : Procesa un pedido y actualiza el inventario y la lista de clientes.

**Código principal:**

1. **Leer el archivo de inventario, el archivo de clientes y el archivo de pedidos.**
2. **Mostrar el menú principal.**
3. **Permitir al usuario seleccionar una opción del menú principal.**
4. **Ejecutar la función correspondiente a la opción seleccionada por el usuario.**
5. **Repetir los pasos 2-4 hasta que el usuario seleccione la opción "Salir".**

**Explicación:**

Este programa es un sistema de control de inventario muy completo que permite al usuario añadir, eliminar y editar artículos en el inventario, así como añadir, eliminar y editar clientes y pedidos. El programa también permite al usuario procesar pedidos, lo que actualiza el inventario y la lista de clientes.

El programa está dividido en varias funciones, cada una de las cuales realiza una tarea específica. Esto hace que el programa sea más fácil de leer y mantener.

El código principal del programa es muy simple. Todo lo que hace es leer los archivos de inventario, clientes y pedidos, mostrar el menú principal y permitir al usuario seleccionar una opción. Una vez que el usuario selecciona una opción, el programa ejecuta la función correspondiente a esa opción.

Las funciones que realizan las tareas específicas del programa son más complejas. Sin embargo, todas están bien documentadas y son fáciles de entender.
