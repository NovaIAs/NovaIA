```scala
// Constantes
val CONSTANTE_1 = "Una constante"
val CONSTANTE_2 = 42

// Variables
var variable_1 = "Una variable"
var variable_2 = 10

// Funciones
def suma(a: Int, b: Int): Int = {
  a + b
}

def resta(a: Int, b: Int): Int = {
  a - b
}

// Clases
class Persona(nombre: String, edad: Int) {
  def getNombre(): String = {
    nombre
  }

  def getEdad(): Int = {
    edad
  }
}

// Objetos
val persona_1 = new Persona("Juan", 20)

// Otros elementos
val lista_1 = List(1, 2, 3, 4, 5)

// Este código es un ejemplo de un código complejo en Scala.
// Contiene constantes, variables, funciones, clases y objetos.

// Las constantes son valores inmutables, que no pueden ser modificados.
// Las variables son valores mutables, que pueden ser modificados.

// Las funciones son bloques de código que reciben parámetros y devuelven un valor.

// Las clases son plantillas para crear objetos.

// Los objetos son instancias de clases.

// Otros elementos, como las listas, son colecciones de datos.

// Este código es un ejemplo de Scala en su forma más compleja.
// Es un lenguaje de programación funcional, que se basa en la idea de que todo es una función.
// Esto significa que los datos y el código son tratados de la misma manera, y que las funciones se pueden combinar entre sí para crear programas más complejos.
```
