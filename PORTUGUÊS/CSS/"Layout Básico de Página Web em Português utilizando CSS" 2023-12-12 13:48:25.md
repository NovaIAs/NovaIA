```css
/* Estilo para o corpo da página */
body {
  font-family: Arial, sans-serif;
  background-color: #f2f2f2;
  margin: 0;
  padding: 0;
}

/* Estilo para o cabeçalho */
header {
  background-color: #333;
  color: #fff;
  padding: 20px;
}

/* Estilo para o título do cabeçalho */
header h1 {
  font-size: 24px;
  margin: 0;
}

/* Estilo para o menu de navegação */
nav {
  background-color: #666;
  padding: 10px;
}

/* Estilo para os itens do menu de navegação */
nav ul {
  list-style-type: none;
  margin: 0;
  padding: 0;
}

nav ul li {
  display: inline-block;
  margin-right: 10px;
}

nav ul li a {
  color: #fff;
  text-decoration: none;
  padding: 5px;
}

nav ul li a:hover {
  background-color: #999;
}

/* Estilo para o conteúdo principal */
main {
  padding: 20px;
}

/* Estilo para as seções do conteúdo principal */
section {
  margin-bottom: 20px;
}

/* Estilo para os títulos das seções */
section h2 {
  font-size: 18px;
  margin: 0;
}

/* Estilo para os parágrafos das seções */
section p {
  line-height: 1.5;
}

/* Estilo para as imagens das seções */
section img {
  max-width: 100%;
  height: auto;
  margin: 10px 0;
}

/* Estilo para o rodapé */
footer {
  background-color: #333;
  color: #fff;
  padding: 20px;
  text-align: center;
}

/* Estilo para o link do rodapé */
footer a {
  color: #fff;
  text-decoration: none;
}

/* Estilo para o hover do link do rodapé */
footer a:hover {
  text-decoration: underline;
}
```

Este código CSS define estilos para um layout básico de uma página web em português. Ele inclui estilos para o corpo da página, cabeçalho, menu de navegação, conteúdo principal, seções, imagens e rodapé.

No estilo do cabeçalho, define-se a cor de fundo e a cor do texto, bem como o preenchimento interno. O título do cabeçalho também tem um estilo específico, com tamanho de fonte e margem definidos.

O menu de navegação tem uma cor de fundo diferente e um preenchimento interno. Os itens do menu são exibidos horizontalmente e têm uma margem direita para separação. Os links do menu têm cor de texto branca e nenhum sublinhado, e ao passar o mouse sobre eles, a cor de fundo muda.

O conteúdo principal tem um preenchimento interno e cada seção dentro dele tem uma margem inferior para separação. Os títulos das seções têm um tamanho de fonte específico e os parágrafos têm um espaçamento de linha definido.

As imagens dentro das seções têm uma largura máxima de 100% e uma altura automática, além de uma margem superior e inferior.

O rodapé tem uma cor de fundo e cor de texto específicas, bem como um preenchimento interno e alinhamento centralizado. O link do rodapé tem cor de texto branca e nenhum sublinhado, e ao passar o mouse sobre ele, o sublinhado é adicionado.
