Claro! Aqui está um exemplo de um código complexo em CSS:

```css
/* Estilo para o body */
body {
  background-color: #f2f2f2;
  font-family: Arial, sans-serif;
  margin: 0;
  padding: 0;
}

/* Estilo para o cabeçalho */
header {
  background-color: #333;
  color: #fff;
  padding: 20px;
}

/* Estilo para o título do cabeçalho */
header h1 {
  font-size: 24px;
  margin: 0;
}

/* Estilo para o menu de navegação */
nav {
  background-color: #666;
  color: #fff;
  padding: 10px;
}

/* Estilo para os links do menu de navegação */
nav a {
  color: #fff;
  margin-right: 10px;
  text-decoration: none;
}

/* Estilo para os links do menu de navegação quando hover */
nav a:hover {
  color: #ff0000;
}

/* Estilo para a seção principal */
section {
  margin: 20px;
}

/* Estilo para os títulos das seções */
section h2 {
  color: #333;
  font-size: 18px;
  margin-bottom: 10px;
}

/* Estilo para as imagens das seções */
section img {
  max-width: 100%;
  height: auto;
}

/* Estilo para a seção de rodapé */
footer {
  background-color: #333;
  color: #fff;
  padding: 20px;
  text-align: center;
}

/* Estilo para o texto do rodapé */
footer p {
  margin: 0;
}

/* Estilo para a classe especial */
.special {
  font-weight: bold;
  color: #ff0000;
}
```

Este código CSS cria estilos para um layout básico de um site. Ele define o estilo para o corpo da página, o cabeçalho, o menu de navegação, as seções, o rodapé e também inclui uma classe especial.

No estilo do cabeçalho, é definida a cor de fundo, a cor do texto e o espaçamento interno. O título dentro do cabeçalho possui um tamanho de fonte específico e margem zero.

O menu de navegação tem uma cor de fundo e cor de texto definidas, assim como um espaçamento interno. Os links dentro do menu de navegação possuem uma margem à direita e a decoração de texto é removida. Quando o mouse passa por cima dos links no menu de navegação, a cor do texto muda para vermelho.

A seção principal tem uma margem externa para posicionamento e contém títulos e imagens. Os títulos das seções possuem uma cor específica, tamanho de fonte e margem inferior. As imagens dentro das seções são responsivas, ou seja, se ajustam ao tamanho do contêiner pai.

O rodapé possui uma cor de fundo, cor do texto e espaçamento interno definidos. O texto dentro do rodapé possui margem zero.

Por fim, existe uma classe especial chamada "special" que torna o texto em negrito e define uma cor vermelha.

Esse código pode ser utilizado como base para estilizar um site e pode ser adaptado de acordo com as necessidades específicas de cada projeto.
