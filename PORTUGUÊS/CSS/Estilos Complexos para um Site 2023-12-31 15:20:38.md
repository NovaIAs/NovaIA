```css
/* Estilos para o corpo do documento */
body {
  background-color: #f2f2f2;
  font-family: Arial, sans-serif;
  margin: 0;
  padding: 0;
}

/* Estilos para o cabeçalho */
header {
  background-color: #333;
  color: #fff;
  padding: 20px;
  text-align: center;
}

header h1 {
  font-size: 24px;
  margin: 0;
}

/* Estilos para a navegação */
nav {
  background-color: #666;
  padding: 10px;
}

nav ul {
  list-style-type: none;
  margin: 0;
  padding: 0;
}

nav ul li {
  display: inline-block;
  margin-right: 10px;
}

nav ul li a {
  color: #fff;
  text-decoration: none;
  padding: 5px 10px;
  border-radius: 3px;
}

nav ul li a:hover {
  background-color: #999;
}

/* Estilos para o conteúdo principal */
main {
  margin: 20px;
}

main h2 {
  font-size: 18px;
  color: #333;
  margin: 0 0 10px;
}

main p {
  line-height: 1.5;
}

/* Estilos para a barra lateral */
aside {
  background-color: #999;
  color: #fff;
  padding: 20px;
  width: 300px;
}

aside h3 {
  font-size: 16px;
  margin: 0 0 10px;
}

aside ul {
  list-style-type: none;
  margin: 0;
  padding: 0;
}

aside ul li {
  margin-bottom: 10px;
}

/* Estilos para o rodapé */
footer {
  background-color: #333;
  color: #fff;
  padding: 20px;
  text-align: center;
}

footer p {
  margin: 0;
}

/* Estilos para a seção de destaque */
.section-highlight {
  background-color: #ffcc00;
  color: #333;
  padding: 20px;
  text-align: center;
}

.section-highlight h2 {
  font-size: 24px;
  margin: 0 0 10px;
}

.section-highlight p {
  font-size: 16px;
  line-height: 1.5;
}
```

Explicação do código:

- O código começa estilizando o corpo do documento, definindo a cor de fundo, a fonte e removendo as margens e preenchimentos padrão (`margin: 0; padding: 0;`).
- Em seguida, estiliza o cabeçalho, definindo a cor de fundo, a cor do texto e o espaço interno (`padding`) de 20 pixels. Também alinha o texto centralmente (`text-align: center;`).
- Define o estilo para a navegação, estabelecendo a cor de fundo e o espaço interno de 10 pixels (`padding`). A lista de itens de navegação é estilizada para remover os pontos de lista padrão (`list-style-type: none;`) e os itens são exibidos em linha (`display: inline-block;`). Os links de navegação têm cor branca (`color: #fff;`), sem decoração de texto (`text-decoration: none;`), preenchimento interno de 5 pixels verticalmente e 10 pixels horizontalmente (`padding: 5px 10px;`), além de bordas arredondadas (`border-radius: 3px;`). Quando o mouse passa por cima dos links, eles têm uma cor de fundo diferente (`background-color: #999;`).
- Os estilos para o conteúdo principal definem a margem externa de 20 pixels e estabelecem um tamanho de fonte de 18 pixels para os títulos (`h2`) e uma altura de linha de 1.5 para os parágrafos (`p`).
- A barra lateral é estilizada com uma cor de fundo cinza escuro (`background-color: #999;`) e um espaço interno de 20 pixels (`padding`). O título (`h3`) tem um tamanho de fonte de 16 pixels e uma margem inferior de 10 pixels. A lista de itens é estilizada para remover os pontos de lista padrão e cada item tem uma margem inferior de 10 pixels.
- O rodapé tem uma cor de fundo escura (`background-color: #333;`) e uma cor de texto branca (`color: #fff;`). O espaço interno é de 20 pixels (`padding`) e o texto é alinhado centralmente (`text-align: center;`).
- A seção de destaque tem uma cor de fundo amarela (`background-color: #ffcc00;`) e uma cor de texto escura (`color: #333;`). O espaço interno é de 20 pixels (`padding`) e o texto é centralizado (`text-align: center;`). O título (`h2`) tem um tamanho de fonte de 24 pixels e uma margem inferior de 10 pixels. Os parágrafos (`p`) têm um tamanho de fonte de 16 pixels e uma altura de linha de 1.5.

Esse código demonstra uma estrutura básica de um site, com estilos para o cabeçalho, navegação, conteúdo principal, barra lateral, rodapé e uma seção de destaque. Claro, esse é apenas um exemplo e você pode personalizar e adaptar os estilos de acordo com suas necessidades.
