Claro! Aqui está um código CSS complexo e diferenciado, que usa seletores avançados e propriedades variadas:

```css
/* Estilizando a página */

html,
body {
  height: 100%;
  margin: 0;
  padding: 0;
  background-color: #f3f3f3;
  font-family: Arial, sans-serif;
}

/* Estilizando o cabeçalho */

header {
  background-color: #333;
  color: #fff;
  padding: 20px;
}

header h1 {
  font-size: 36px;
  margin: 0;
}

header p {
  font-size: 18px;
  margin: 10px 0;
}

/* Estilizando a navegação */

nav {
  background-color: #666;
  padding: 10px;
}

nav ul {
  list-style-type: none;
  margin: 0;
  padding: 0;
  display: flex;
  justify-content: space-between;
}

nav li {
  margin-right: 10px;
}

nav a {
  text-decoration: none;
  color: #fff;
  padding: 10px;
}

/* Estilizando o conteúdo principal */

main {
  padding: 20px;
}

main h2 {
  color: #333;
  font-size: 24px;
}

main p {
  color: #555;
  font-size: 16px;
  line-height: 1.5;
  margin-bottom: 10px;
}

main .destaque {
  color: #ff0000;
  font-weight: bold;
}

/* Estilizando a seção de formulário */

form {
  background-color: #fff;
  padding: 20px;
  border: 1px solid #ccc;
  border-radius: 5px;
}

form label {
  display: block;
  margin-bottom: 10px;
  font-weight: bold;
}

form input[type="text"],
form input[type="email"],
form textarea {
  width: 100%;
  padding: 10px;
  border: 1px solid #ccc;
  border-radius: 3px;
  margin-bottom: 10px;
}

form button {
  background-color: #333;
  color: #fff;
  padding: 10px 20px;
  border: none;
  border-radius: 3px;
}

/* Estilizando o rodapé */

footer {
  background-color: #333;
  color: #fff;
  padding: 20px;
  text-align: center;
}

/* Estilizando os links */

a {
  color: #333;
  transition: color 0.3s;
}

a:hover {
  color: #ff0000;
}
```

Este é um código CSS que estiliza uma página web completa. Explicando alguns pontos-chave do código:

- Definimos o estilo da página como um todo, definindo a cor de fundo, a fonte e o tamanho do texto.
- Estilizamos o cabeçalho com um fundo escuro, cor branca e tamanho de fonte diferente para o título e o parágrafo.
- Estilizamos a navegação com um fundo mais claro, usando flexbox para alinhar os itens e aplicamos um espaçamento entre eles.
- Estilizamos o conteúdo principal com tamanhos de fonte e cores diferentes para os títulos e parágrafos, destacando um texto com a cor vermelha e negrito.
- Estilizamos um formulário com um fundo branco, bordas arredondadas e espaçamento adequado.
- Por fim, estilizamos o rodapé com um fundo escuro, cor branca e alinhamento centralizado.

Esse código é apenas um exemplo de como o CSS pode ser usado para estilizar uma página web. Existem muitas outras propriedades e recursos disponíveis, e o código pode ser personalizado de acordo com as necessidades específicas do projeto.
