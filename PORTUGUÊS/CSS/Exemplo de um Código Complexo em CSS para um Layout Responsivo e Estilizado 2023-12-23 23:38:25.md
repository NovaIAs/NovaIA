Claro! Aqui está um exemplo de um código complexo em CSS:

```css
/* Estilizando o corpo da página */
body {
  background-color: #f2f2f2;
  font-family: Arial, sans-serif;
  margin: 0;
  padding: 0;
}

/* Estilizando o cabeçalho */
header {
  background-color: #333;
  color: #fff;
  padding: 20px;
}

header h1 {
  font-size: 24px;
  margin: 0;
}

/* Estilizando o menu de navegação */
nav {
  background-color: #666;
  padding: 10px;
}

nav ul {
  list-style-type: none;
  margin: 0;
  padding: 0;
}

nav ul li {
  display: inline;
  margin-right: 10px;
}

nav ul li a {
  color: #fff;
  text-decoration: none;
  padding: 5px;
}

/* Estilizando a seção principal */
main {
  margin: 20px;
}

section {
  background-color: #fff;
  border: 1px solid #ccc;
  border-radius: 5px;
  padding: 20px;
  margin-bottom: 20px;
}

section h2 {
  font-size: 18px;
  margin: 0 0 10px 0;
}

section p {
  line-height: 1.5;
}

/* Estilizando a barra lateral */
aside {
  width: 300px;
  float: right;
  background-color: #f2f2f2;
  padding: 20px;
  margin-left: 20px;
}

/* Estilizando o rodapé */
footer {
  background-color: #333;
  color: #fff;
  padding: 20px;
}

footer p {
  margin: 0;
}

/* Estilizando as classes especiais */
.destaque {
  background-color: yellow;
}

.marcado {
  text-decoration: underline;
}

.animado {
  animation: animacao 5s linear infinite;
}

@keyframes animacao {
  0% {
    opacity: 0;
    transform: translateY(-20px);
  }
  50% {
    opacity: 1;
    transform: translateY(0);
  }
  100% {
    opacity: 0;
    transform: translateY(20px);
  }
}
```

Este código complexo em CSS é projetado para criar um layout responsivo e estilizado para uma página web. Ele inclui estilos para o corpo da página, cabeçalho, menu de navegação, seção principal, barra lateral e rodapé.

O código começa definindo o estilo para o corpo da página, definindo a cor de fundo, a fonte utilizada e removendo margens e preenchimentos padrão. A seguir, define o estilo para o cabeçalho, aplicando uma cor de fundo escura, cor de texto clara e espaçamento interno.

O menu de navegação é estilizado com uma cor de fundo mais clara e uma lista horizontal de itens de menu.

A seção principal é definida com uma margem externa e um estilo de caixa de conteúdo com uma cor de fundo branca, borda cinza e bordas arredondadas.

A barra lateral é estilizada com uma largura fixa, cor de fundo clara e margem esquerda.

O rodapé é estilizado com uma cor de fundo escura, cor de texto clara e espaçamento interno.

Em seguida, temos três classes especiais: "destaque", "marcado" e "animado". A classe "destaque" aplica uma cor de fundo amarela para destacar elementos específicos. A classe "marcado" aplica um sublinhado aos elementos. A classe "animado" aplica uma animação de fade-in e fade-out aos elementos com a duração de 5 segundos.

Por fim, há um conjunto de keyframes que definem a animação "animacao" utilizada pela classe "animado".

Este código complexo em CSS visa demonstrar uma variedade de estilos e técnicas para a criação de um layout visualmente atraente e interativo.
