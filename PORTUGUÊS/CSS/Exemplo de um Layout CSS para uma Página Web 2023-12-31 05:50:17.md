Claro! Aqui está um exemplo de um código complexo em CSS que utiliza várias propriedades e seletores diferentes:

```css
/* Estilo para o corpo do documento */
body {
  font-family: Arial, sans-serif;
  background-color: #f2f2f2;
  margin: 0;
  padding: 0;
}

/* Estilo para o cabeçalho */
header {
  background-color: #333;
  color: #fff;
  padding: 20px;
}

/* Estilo para o título do cabeçalho */
header h1 {
  font-size: 24px;
  margin: 0;
}

/* Estilo para o menu de navegação */
nav {
  background-color: #666;
  padding: 10px;
}

/* Estilo para os links do menu de navegação */
nav a {
  color: #fff;
  text-decoration: none;
  margin-right: 10px;
}

/* Estilo para os links do menu de navegação quando estão sendo hover */
nav a:hover {
  color: #ff0000;
}

/* Estilo para a seção principal */
section {
  margin: 20px;
}

/* Estilo para os blocos de conteúdo */
div.block {
  background-color: #fff;
  padding: 10px;
  margin-bottom: 20px;
}

/* Estilo para as imagens dentro dos blocos de conteúdo */
div.block img {
  max-width: 100%;
  height: auto;
  margin-bottom: 10px;
}

/* Estilo para os títulos dentro dos blocos de conteúdo */
div.block h2 {
  font-size: 18px;
  margin: 0;
}

/* Estilo para os parágrafos dentro dos blocos de conteúdo */
div.block p {
  line-height: 1.5;
}

/* Estilo para o rodapé */
footer {
  background-color: #333;
  color: #fff;
  padding: 20px;
  text-align: center;
}

/* Estilo para os links no rodapé */
footer a {
  color: #fff;
  text-decoration: none;
}

/* Estilo para os links no rodapé quando estão sendo hover */
footer a:hover {
  color: #ff0000;
}
```

Este código CSS cria um layout básico para uma página web. Ele define estilos para o corpo do documento, cabeçalho, menu de navegação, seção principal, blocos de conteúdo, imagens, títulos, parágrafos e rodapé.

O código utiliza seletores como `body`, `header`, `nav`, `section`, `div`, além de seletores aninhados como `header h1` para estilizar o título do cabeçalho e `div.block img` para estilizar as imagens dentro dos blocos de conteúdo.

Também são utilizadas diversas propriedades como `font-family`, `background-color`, `color`, `padding`, `margin`, `text-decoration`, `line-height`, `max-width`, `height`, entre outras, para definir o estilo de cada elemento.

Espero que este exemplo atenda às suas expectativas!
