Claro! Aqui está um exemplo de um código CSS complexo e diferenciado:

```css
/* Estilos para o corpo do documento */
body {
  font-family: Arial, sans-serif;
  background-color: #f6f6f6;
  margin: 0;
  padding: 0;
}

/* Estilos para os títulos */
h1 {
  font-size: 32px;
  color: #333;
  margin-bottom: 20px;
}

h2 {
  font-size: 24px;
  color: #555;
  margin-bottom: 15px;
}

/* Estilos para as seções */
.section {
  padding: 20px;
  background-color: #fff;
  box-shadow: 0 2px 5px rgba(0, 0, 0, 0.1);
  margin-bottom: 30px;
}

.section-heading {
  font-size: 20px;
  color: #777;
  margin-bottom: 10px;
}

/* Estilos para as listas */
ul {
  list-style: none;
  padding: 0;
  margin: 0;
}

ul li {
  padding: 5px 0;
  border-bottom: 1px solid #eee;
}

/* Estilos para os botões */
.button {
  display: inline-block;
  padding: 10px 20px;
  font-size: 16px;
  color: #fff;
  background-color: #007bff;
  border: none;
  border-radius: 5px;
  cursor: pointer;
}

.button:hover {
  background-color: #0056b3;
}

/* Estilos para as imagens */
img {
  max-width: 100%;
  height: auto;
}

/* Estilos para as tabelas */
.table {
  width: 100%;
  border-collapse: collapse;
}

.table thead th {
  font-weight: bold;
  background-color: #f0f0f0;
  padding: 10px;
  text-align: left;
  border-bottom: 1px solid #ccc;
}

.table tbody td {
  padding: 10px;
  border-bottom: 1px solid #eee;
}
```

Neste exemplo, temos estilos para o corpo do documento, títulos, seções, listas, botões, imagens e tabelas. Utilizei elementos como `body`, `h1`, `h2`, `.section`, `.section-heading`, `ul`, `li`, `.button`, `img`, `.table` para aplicar estilos específicos a cada parte do código.

Os estilos foram definidos para fonte, cor, margens, espaçamentos, sombras, fundos, bordas, tamanhos, entre outros detalhes, proporcionando um código visualmente agradável e organizado.

Espero que esse código atenda às suas expectativas!
