Claro! Aqui está um código CSS complexo para você:

```css
/* Estilo para o corpo do documento */
body {
  font-family: Arial, sans-serif;
  background-color: #f1f1f1;
  margin: 0;
  padding: 0;
}

/* Estilo para o cabeçalho */
header {
  background-color: #333;
  color: #fff;
  padding: 20px;
  text-align: center;
}

/* Estilo para a barra de navegação */
nav {
  background-color: #555;
  padding: 10px;
}

/* Estilo para os links na barra de navegação */
nav a {
  color: #fff;
  text-decoration: none;
  margin-right: 20px;
}

/* Estilo para os links quando estão ativos */
nav a.active {
  font-weight: bold;
}

/* Estilo para a seção principal */
section {
  margin: 20px;
  padding: 20px;
  background-color: #fff;
  border-radius: 5px;
}

/* Estilo para os títulos */
h1, h2, h3 {
  color: #333;
}

h1 {
  font-size: 24px;
}

h2 {
  font-size: 20px;
}

h3 {
  font-size: 16px;
}

/* Estilo para a seção de rodapé */
footer {
  background-color: #333;
  color: #fff;
  padding: 20px;
  text-align: center;
}

/* Estilo para as imagens */
img {
  max-width: 100%;
  height: auto;
}

/* Estilo para uma tabela */
table {
  border-collapse: collapse;
  width: 100%;
}

/* Estilo para as células da tabela */
th, td {
  padding: 8px;
  text-align: left;
  border-bottom: 1px solid #ddd;
}

/* Estilo para as linhas alternadas da tabela */
tr:nth-child(even) {
  background-color: #f2f2f2;
}

/* Estilo para os botões */
button {
  background-color: #4CAF50;
  color: white;
  padding: 10px 20px;
  border: none;
  border-radius: 4px;
  cursor: pointer;
}

/* Estilo para os botões quando passa o mouse por cima */
button:hover {
  background-color: #45a049;
}

/* Estilo para os botões quando pressionados */
button:active {
  background-color: #3e8e41;
}

/* Estilo para as caixas de texto */
input[type=text] {
  width: 100%;
  padding: 12px 20px;
  margin: 8px 0;
  box-sizing: border-box;
  border: 1px solid #ccc;
  border-radius: 4px;
}

/* Estilo para as caixas de seleção */
select {
  width: 100%;
  padding: 12px 20px;
  margin: 8px 0;
  box-sizing: border-box;
  border: 1px solid #ccc;
  border-radius: 4px;
}

/* Estilo para as mensagens de erro */
.error {
  color: red;
  font-weight: bold;
}
```

Este código contém uma série de estilos CSS para diferentes elementos de uma página web. Aqui estão algumas explicações:

- O estilo para o corpo do documento define a fonte, cor de fundo e margens.
- O estilo para o cabeçalho define a cor de fundo, cor do texto e espaçamento interno.
- O estilo para a barra de navegação define a cor de fundo e espaçamento interno.
- O estilo para os links na barra de navegação define a cor do texto e margem direita.
- O estilo para os links ativos define um negrito para destacá-los.
- O estilo para a seção principal define margens, espaçamento interno, cor de fundo e borda arredondada.
- Os estilos para os títulos definem tamanhos de fonte diferentes.
- O estilo para a seção de rodapé define a cor de fundo, cor do texto e espaçamento interno.
- O estilo para as imagens define uma largura máxima de 100% e altura automática.
- O estilo para tabelas define a largura de 100% e o colapso da borda.
- O estilo para células e linhas alternadas da tabela define o preenchimento, alinhamento e borda inferior.
- O estilo para os botões define a cor de fundo, cor do texto, espaçamento interno, borda e cursor.
- Os estilos para os botões:hover e :active modificam a cor de fundo quando o mouse está em cima ou quando o botão é pressionado.
- O estilo para as caixas de texto e seleção define a largura, preenchimento, margem, borda e borda arredondada.
- O estilo para mensagens de erro define a cor do texto e negrito para destacá-las.

Espero que este código atenda às suas expectativas!
