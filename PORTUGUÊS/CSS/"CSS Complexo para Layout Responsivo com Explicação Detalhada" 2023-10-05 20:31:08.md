  font-size: 24px;
  margin: 0;
}

/* ============ Menu ============ */

/* Define o estilo do menu de navegação */

  background-color: #666;

  padding: 20px;

Explicação:

1. O estilo global define elementos compartilhados em todo o documento HTML, como a fonte, a cor de fundo do body e a margem.

2. O cabeçalho (`header`) possui um estilo específico, com uma cor de fundo escura (`#333`), texto branco (`#fff`) e um espaçamento interno de 20 pixels.

3. O título (`h1`) do cabeçalho possui um tamanho de fonte de 24 pixels e margem zero.

4. O menu de navegação (`nav`) tem uma cor de fundo cinza escuro (`#666`), texto branco (`#fff`) e um espaçamento interno de 10 pixels.

5. A lista não ordenada (`ul`) dentro do menu não possui marcadores de lista e tem margem e preenchimento zero.

6. Cada item da lista (`li`) do menu é exibido lado a lado (`inline`) e tem uma margem direita de 10 pixels.

7. Os links (`a`) dentro de cada item do menu têm cor branca e nenhum sublinhado, com um preenchimento de 5 pixels. Quando o mouse passa sobre eles, a cor de fundo muda para cinza claro (`#999`).

8. O conteúdo principal (`main`) tem um espaçamento interno de 20 pixels.

9. Os títulos (`h2`) do conteúdo têm um tamanho de fonte de 18 pixels e margem superior zero.

10. Os parágrafos (`p`) do conteúdo têm uma altura de linha de 1.5 (espaçamento entre linhas).

11. O rodapé (`footer`) possui um estilo semelhante ao cabeçalho, com uma cor de fundo escura, texto branco e espaçamento interno de 20 pixels. O texto é centralizado.

12. Os links (`a`) dentro do rodapé têm cor branca e nenhum sublinhado. Quando o mouse passa sobre eles, o sublinhado é aplicado.
