Claro! Aqui está um exemplo de um código CSS complexo e diferenciado que utiliza vários recursos e propriedades:

```css
/* Estilo do corpo da página */
body {
  font-family: Arial, sans-serif;
  background-color: #f5f5f5;
  margin: 0;
  padding: 0;
}

/* Estilo do cabeçalho da página */
header {
  background-color: #333;
  color: #fff;
  padding: 20px;
}

header h1 {
  font-size: 24px;
  margin: 0;
}

/* Estilo da barra de navegação */
nav {
  background-color: #666;
  padding: 10px;
}

nav ul {
  list-style-type: none;
  margin: 0;
  padding: 0;
}

nav ul li {
  display: inline-block;
  margin-right: 10px;
}

nav ul li a {
  text-decoration: none;
  color: #fff;
  padding: 5px;
}

/* Estilo do conteúdo principal */
main {
  padding: 20px;
}

main h2 {
  font-size: 20px;
  margin: 0;
}

main p {
  line-height: 1.5;
}

/* Estilo da seção de destaque */
section.highlight {
  background-color: #ffcc00;
  padding: 20px;
  margin-bottom: 20px;
}

section.highlight h3 {
  font-size: 18px;
  margin: 0;
}

section.highlight p {
  font-style: italic;
}

/* Estilo do rodapé da página */
footer {
  background-color: #333;
  color: #fff;
  padding: 20px;
  text-align: center;
}

footer p {
  margin: 0;
}

/* Estilo de um botão personalizado */
.button {
  display: inline-block;
  background-color: #3399ff;
  color: #fff;
  padding: 10px 20px;
  border-radius: 5px;
  text-decoration: none;
  text-transform: uppercase;
  transition: background-color 0.3s ease;
}

.button:hover {
  background-color: #0080ff;
}
```

Este código CSS cria estilos para diferentes elementos de uma página web. O corpo da página tem uma cor de fundo, o cabeçalho e o rodapé têm cores diferentes, a barra de navegação é estilizada com uma lista de itens, o conteúdo principal possui títulos e parágrafos, e há uma seção de destaque com um estilo especial.

Além disso, o código também define um estilo para um botão personalizado, que muda de cor quando o cursor passa por cima dele.

Espero que este exemplo tenha sido útil!
