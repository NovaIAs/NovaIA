Claro! Aqui está um código CSS complexo e diferenciado, que utiliza diferentes propriedades para estilizar elementos de uma página web:

```css
/* Estilizando o corpo da página */
body {
    background-color: #f2f2f2;
    font-family: Arial, sans-serif;
    margin: 0;
    padding: 0;
}

/* Estilizando o cabeçalho */
header {
    background-color: #333;
    color: #fff;
    padding: 20px;
    text-align: center;
}

header h1 {
    font-size: 36px;
    margin: 0;
}

/* Estilizando a navegação */
nav {
    background-color: #666;
    padding: 10px;
}

nav ul {
    list-style-type: none;
    margin: 0;
    padding: 0;
}

nav ul li {
    display: inline-block;
    margin-right: 10px;
}

nav ul li a {
    color: #fff;
    text-decoration: none;
    padding: 6px 10px;
}

nav ul li a:hover {
    background-color: #999;
}

/* Estilizando o conteúdo principal */
main {
    margin: 20px;
}

/* Estilizando as seções */
section {
    background-color: #fff;
    border: 1px solid #ccc;
    border-radius: 4px;
    margin-bottom: 20px;
    padding: 20px;
}

section h2 {
    font-size: 24px;
    margin-top: 0;
}

section p {
    font-size: 16px;
    line-height: 1.5;
}

/* Estilizando as caixas de destaque */
.box {
    background-color: #f9f9f9;
    border: 1px solid #ccc;
    border-radius: 4px;
    padding: 10px;
}

.box p {
    font-size: 14px;
}

    background-color: #333;
    color: #fff;
    padding: 20px;
    text-align: center;
}

    margin: 0;
}
```

Neste código, estou estilizando diferentes elementos de uma página web. Começando pelo corpo da página, defino a cor de fundo, a fonte utilizada e removo as margens e espaçamentos indesejados. Em seguida, estilizo o cabeçalho, a navegação, o conteúdo principal, as seções, as caixas de destaque e o rodapé.

Utilizo propriedades como `background-color`, `color`, `padding`, `text-align`, `font-size`, `margin`, `border`, `border-radius` e `line-height` para criar diferentes efeitos visuais.
